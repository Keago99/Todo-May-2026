# Todo App Color Scheme

| Element | Color | Hex |
|---------|-------|-----|
| Background | Dark gray | `#1e1e2e` |
| Sidebar | Darker gray | `#181825` |
| Card background | Medium-dark gray | `#313244` |
| Primary text | Off-white | `#cdd6f4` |
| Secondary text | Muted gray | `#6c7086` |
| Accent / buttons | Purple or Blue | `#89b4fa` or `#cba6f7` |
| High priority | Red-orange | `#f38ba8` |
| Completed task | Muted green | `#a6e3a1` |
